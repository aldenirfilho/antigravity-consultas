# 🛡️ Segurança e Curadoria

## Regras de segurança

Todo conteúdo médico deve conter, quando aplicável:

- Indicações.
- Contraindicações.
- Riscos.
- Monitorização.
- Ajustes por função renal/hepática.
- Alertas de dose.
- Quando chamar especialista.
- Quando transferir.
- Quando considerar UTI.

## Marcação de conteúdo

- `📌 Ponto prático` — conduta aplicável.
- `🏆 Ponto TEMI` — alta chance de prova.
- `⚠️ Pegadinha` — erro comum ou risco.
- `🔎 Informação extra` — conteúdo externo ao material principal.
- `🧠 Fisiopatologia nerd` — mecanismo profundo.
- `🛡️ Segurança` — ponto de prevenção de dano.

## Revisão recomendada

1. Revisão por médico responsável.
2. Revisão de fonte.
3. Revisão de doses.
4. Revisão de links.
5. Revisão de atualização.
