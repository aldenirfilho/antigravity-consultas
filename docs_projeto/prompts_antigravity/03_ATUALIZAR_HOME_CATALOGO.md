# 03 — Atualizar Home com Catálogo

Atualize `index.html` para criar uma seção:

`📦 Catálogo do Projeto`

Requisitos:

1. Carregar `data/catalogo.json`.
2. Renderizar cards responsivos.
3. Cada card deve mostrar:
   - título;
   - legenda;
   - status;
   - tipo;
   - tema;
   - botão “Abrir”.
4. Se o JSON falhar, renderizar fallback mínimo.
5. Preservar estilo visual da home atual.
6. Não quebrar os scripts existentes:
   - `assets/js/app.js`
   - `assets/js/search.js`
   - `assets/js/graph.js`
