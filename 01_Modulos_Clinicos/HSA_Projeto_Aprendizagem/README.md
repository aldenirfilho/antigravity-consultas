# 🧠 HSA Aneurismática — Projeto de Aprendizagem

## 📋 Sobre
App web educacional interativo sobre **Hemorragia Subaracnóidea Aneurismática: Tratamento e Prognóstico**, baseado no artigo do UpToDate (Tópico 1127, v52.0, 2026).

## 🎯 Funcionalidades
| Aba | Conteúdo |
|-----|----------|
| 📋 Resumo | Síntese completa do artigo com checklists interativas |
| 📊 Escalas | Hunt-Hess, WFNS, Fisher, Fisher Modificada, mRS |
| 🔀 Fluxo | Fluxograma visual de manejo desde admissão até alta |
| 🃏 Cards | 20 flashcards com flip 3D para revisão espaçada |
| ❓ Quiz | 20 questões estilo TEMI/AMIB com feedback detalhado |

## 🚀 Como usar
1. Abrir `index.html` no navegador (Chrome/Safari)
2. Navegar pelas abas
3. Usar checklists para conferir protocolos
4. Revisar flashcards diariamente
5. Refazer quiz até alcançar ≥80%

## ⚠️ Limitações
- Ferramenta **educacional/de apoio** — não substitui julgamento clínico
- Baseado em artigo único do UpToDate — consultar protocolos institucionais
- Dados fictícios — sem informações de pacientes reais

## 📚 Fonte
- Singer RJ, Ogilvy CS, Rordorf G. *Aneurysmal subarachnoid hemorrhage: Treatment and prognosis.* UpToDate 2026. Tópico 1127, v52.0.
