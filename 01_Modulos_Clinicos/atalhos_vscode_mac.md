## 🗂️ 1. Navegação de Interface e Arquivos
| Atalho | Função |
| :--- | :--- |
| `Cmd + Shift + P` | Abrir Paleta de Comandos (Ação Principal) |
| `Cmd + P` | Busca rápida de arquivos no projeto |
| `Cmd + B` | Alternar Barra Lateral Esquerda (Arquivos) |
| `Cmd + Option + B` | Alternar Barra Lateral Direita (Agente IA / Antigravity) |
| `Cmd + J` | Alternar Painel Inferior (Terminal/Output) |
| `Cmd + 1 / 2 / 3` | Focar no grupo de editores 1, 2 ou 3 |
| `Cmd + Option + ← / →` | Alternar entre abas abertas |
| `Cmd + Shift + T` | Reabrir a última aba fechada |
| `Cmd + W` | Fechar aba atual |
| `Cmd + K, Z` | Modo Zen (Foco Total, oculta interface) |
| `Cmd + K, S` | Salvar tudo |

## ✍️ 2. Edição de Texto TDAH-Friendly
| Atalho | Função |
| :--- | :--- |
| `Cmd + X` | Recortar linha inteira (sem precisar selecionar nada) |
| `Cmd + C` | Copiar linha inteira (sem precisar selecionar nada) |
| `Option + ↑ / ↓` | Mover a linha atual para cima ou para baixo |
| `Shift + Option + ↑ / ↓` | Duplicar a linha atual para cima ou para baixo |
| `Cmd + Shift + K` | Excluir linha atual completamente |
| `Cmd + Enter` | Inserir nova linha em branco abaixo |
| `Cmd + Shift + Enter` | Inserir nova linha em branco acima |
| `Option + Z` | Alternar quebra de linha visual (Word Wrap) |
| `Cmd + /` | Comentar ou Descomentar a linha atual |
| `Shift + Option + A` | Comentar ou Descomentar bloco de código |

## 🖱️ 3. Multi-cursores e Seleção (Magia do VS Code)
| Atalho | Função |
| :--- | :--- |
| `Option + Click` | Inserir múltiplos cursores manuais |
| `Cmd + Option + ↑ / ↓` | Adicionar um cursor exatamente na linha de cima ou de baixo |
| `Cmd + D` | Selecionar a próxima ocorrência da mesma palavra |
| `Cmd + Shift + L` | Selecionar TODAS as ocorrências da palavra atual de uma vez |
| `Shift + Option + I` | Adicionar cursor no final de todas as linhas que estão selecionadas |
| `Cmd + I` | Selecionar a linha inteira |
| `Shift + Option + →` | Expandir seleção de forma inteligente (pega palavras, depois blocos, depois funções) |
| `Shift + Option + ←` | Encolher seleção inteligente |

## 🔍 4. Busca, Substituição e Refatoração
| Atalho | Função |
| :--- | :--- |
| `Cmd + F` | Buscar texto no arquivo atual |
| `Cmd + Option + F` | Substituir texto no arquivo atual |
| `Cmd + Shift + F` | Buscar texto em TODOS os arquivos do projeto |
| `Cmd + Shift + H` | Substituir texto em TODOS os arquivos do projeto |
| `Cmd + G` | Encontrar a próxima ocorrência na busca |
| `F2` | Renomear símbolo/variável em todo o projeto com segurança |
| `Shift + Option + F` | Formatar o documento inteiro (Alinha o código automaticamente) |
| `F12` | Ir para a Definição (ex: ver o que uma função médica faz) |
| `Option + F12` | Espiar a Definição (Abre um popup flutuante rápido sem sair da tela atual) |
| `Shift + F12` | Encontrar todas as referências (onde essa função é usada) |

## 💻 5. Terminal e Controle de Erros
| Atalho | Função |
| :--- | :--- |
| `Ctrl + \`` | Alternar visibilidade do Terminal Integrado (abre e fecha) |
| `Ctrl + Shift + \`` | Criar nova aba de Terminal Integrado |
| `Cmd + ↑ / ↓` | (Dentro do terminal) Rolar histórico de mensagens |
| `F8` | Pular direto para o próximo Erro de sintaxe ou Aviso (Linter) |
| `Shift + F8` | Voltar para o Erro ou Aviso anterior |
| `Cmd + .` | Ações Rápidas (Sugestões de correção da IDE) |

## 🍎 6. Mac OS (Produtividade Geral)
| Atalho | Função |
| :--- | :--- |
| `Cmd + Space` | Busca Global do Spotlight |
| `Cmd + Tab` | Alternar rapidamente entre aplicativos abertos |
| `Cmd + Shift + 3` | Print da tela inteira |
| `Cmd + Shift + 4` | Print de área selecionada |
| `Cmd + Shift + 4, depois Espaço` | Print de janela específica (imagem limpa, sem fundo) |
| `Cmd + Option + Esc` | Forçar encerramento de aplicativo travado (Force Quit) |
| `Cmd + H` | Esconder aplicativo atual (Limpar tela para manter foco) |
| `Ctrl + ↑` | Mission Control (Ver todas as janelas abertas) |
| `Ctrl + ← / →` | Alternar entre Spaces (Telas virtuais do Mac) |

## 🤖 7. IA e Agentes (Antigravity/Copilot)
| Atalho | Função |
| :--- | :--- |
| `Cmd + L` ou `Cmd + Shift + L` | Focar na janela principal do Agente de IA |
| `Cmd + I` | Iniciar Chat Inline de Código (Pequena janela sobre o código) |
| `Tab` | Aceitar sugestão de autocompletar em cinza da IA |
| `Cmd + →` | Aceitar apenas a próxima palavra da sugestão da IA |
| `Option + ]` | Alternar para a próxima sugestão da IA (se houver mais de uma) |
| `Option + [` | Alternar para a sugestão anterior da IA |

---
> [!TIP]
> **Como memorizar (Foco TDAH):** Não tente aprender todos de uma vez. Pegue 2 por dia. Comece com a mágica do `Cmd + D` (multi-seleção) e o `Cmd + P` (busca de arquivos).
