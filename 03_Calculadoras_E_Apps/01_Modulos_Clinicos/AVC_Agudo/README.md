# App Clínico: AVC Agudo (Manejo e Tratamento)

## 📌 Objetivo
Aplicativo web mobile-first criado para auxiliar médicos (foco em plantão, UTI/Emergência e preparação TEMI) no manejo rápido e baseado em diretrizes do Acidente Vascular Cerebral Isquêmico Agudo. 

## ⚠️ Segurança Clínica
**Isenção de Responsabilidade:** Esta é uma ferramenta educacional e de apoio à decisão clínica desenvolvida para o Dr. Aldenir Rocha. **NÃO substitui o julgamento clínico individual ou protocolos institucionais.** Todos os dados baseiam-se na revisão *Avaliação inicial e tratamento do AVC agudo* (UpToDate). Nunca utilize dados reais e identificáveis de pacientes.

## 🚀 Funcionalidades
- **Triagem Rápida:** Escalas FAST e LAMS para Oclusão de Grande Vaso (LVO).
- **Manejo Agudo (Fast Track):** Metas de Pressão Arterial, Glicemia, Hipóxia e indicações de Trombolítico/Trombectomia.
- **Diferenciais (Mimics):** Diagnósticos alternativos a descartar rapidamente.
- **Flashcards:** Treinamento ativo com perguntas padrão TEMI/AMIB.
