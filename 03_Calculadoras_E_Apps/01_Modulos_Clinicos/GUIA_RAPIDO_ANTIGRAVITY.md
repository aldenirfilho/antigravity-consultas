# 🚀 Guia Rápido Antigravity — Modo TDAH & Intensivista

Este guia foi criado para otimizar o seu uso do **Google Antigravity** e garantir a máxima produtividade e segurança, respeitando seu perfil (Intensivista, TDAH, foco em eficiência).

---

## 🧠 1. Regras de Ouro para Render Mais

- **Seja Direto:** Pule "por favor" e "obrigado". Dê a ordem clara.
- **Quebre o Problema:** Se a tarefa for grande, peça: *"Divida essa tarefa em 3 passos pequenos"*.
- **Exija Formato Visual:** Peça sempre tabelas, checklists ou marcadores (bullets). Textos longos são inimigos do foco.
- **Use "Aprove ou Rejeite":** Antes de eu modificar arquivos vitais, peça para eu gerar um **Plano de Ação (Implementation Plan)** e espere sua aprovação.
- **Evite Context Switching:** Trabalhe em **um problema por vez**. Não misture refatoração de código com pesquisa médica no mesmo prompt se não for necessário.

---

## 🛠️ 2. Melhores Prompts (Comandos) para Usar

Copie e cole essas estruturas para extrair o melhor de mim:

### 💻 Para Programação e Projetos (Checklist UTI, etc)
- *"Crie uma estrutura inicial para um app de [TEMA]. Faça apenas o MVP e liste os arquivos que serão criados."*
- *"Refatore este código para deixá-lo mais legível. Adicione comentários em português focados na lógica."*
- *"Ache o erro (bug) neste código e me explique o motivo em 2 linhas antes de corrigir."*
- *"Crie um design Mobile-First (pensando em iPhone) para esta funcionalidade. Use botões grandes e leitura rápida."*

### 🏥 Para Apoio Clínico / TEMI
- *"Resuma o protocolo de [DOENÇA/DIRETRIZ] em uma tabela focada em Condutas Imediatas (SBAR)."*
- *"Gere 5 flashcards curtos e difíceis estilo TEMI sobre [TEMA]. Formato: Pergunta curta / Resposta em tópicos."*
- *"Crie um fluxograma mental (passo a passo numerado) para tomada de decisão em caso de [SINTOMA/DIAGNÓSTICO]."*

### 🎯 Para Organização e Produtividade Pessoal
- *"Analise esta lista de tarefas e me diga qual devo fazer primeiro usando a Matriz de Eisenhower."*
- *"Estou travado na tarefa X. Me dê o primeiro passo prático (micro-passo) de 5 minutos para eu começar agora."*
- *"Organize esses dados embaralhados em uma tabela clara com colunas: Data, Valor e Categoria."*

## ⚙️ 3. Como o Antigravity pode Agir (Ferramentas que possuo)

Eu não sou só um chat! Eu posso executar ações reais na sua máquina (sempre pedindo permissão quando perigoso):

1. **Editar e Criar Arquivos:** Posso codar direto nos arquivos do seu projeto.
2. **Navegar na Web (`search_web`):** Posso buscar documentações técnicas atualizadas.
3. **Executar Comandos (`run_command`):** Posso rodar testes, instalar pacotes npm, criar pastas, etc. (Sempre te avisarei o risco).
4. **Ler Arquivos (`view_file`):** Posso ler logs de erro gigantes e te resumir só o que importa.
5. **Criar Artifacts (Documentos Especiais):** Para planos longos, uso documentos bonitos em markdown para não poluir o chat.

---

## ⌨️ 4. Principais Atalhos de Teclado (VS Code + AI)

Esses são os atalhos de teclado que vão te poupar muito tempo ao usar o Antigravity no VS Code (no Mac):

- `Cmd + L` : Abre a janela lateral do chat do Antigravity (para iniciar uma conversa nova ou ver as respostas).
- `Cmd + I` : Abre o chat *inline* (uma janelinha que flutua direto em cima do código que você está escrevendo).
- `Cmd + Shift + P` : Abre a paleta de comandos geral do VS Code.
- `Cmd + B` : Esconde/Mostra a barra lateral (File Explorer) para te dar mais foco na tela.
- `Cmd + J` : Esconde/Mostra o painel inferior (onde fica o Terminal).

---

## 🚨 5. Lembretes de Segurança

- **NUNCA** cole dados reais de pacientes (LGPD). Use *Paciente X*.
- **PERGUNTE** antes de deletar algo ("Tem certeza que quer apagar a pasta X?").
- **DICA:** Se eu começar a responder algo muito longo ou fora do foco, mande um **"Pare. Seja mais objetivo e volte ao ponto X"**.
