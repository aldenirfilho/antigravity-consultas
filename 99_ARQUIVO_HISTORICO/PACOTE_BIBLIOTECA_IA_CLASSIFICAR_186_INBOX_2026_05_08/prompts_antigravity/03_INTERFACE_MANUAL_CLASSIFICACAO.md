# 03 — Criar/aperfeiçoar interface manual de classificação

Atualizar `05_Biblioteca_IA/index.html` para incluir painel manual de triagem:

Campos:

- preview do arquivo;
- tema;
- status;
- título;
- tags;
- resumo;
- uso clínico;
- relevância TEMI;
- alerta de segurança;
- conexões cerebrais;
- botão copiar JSON;
- botão exportar catálogo.

Observação:
GitHub Pages estático não grava direto no repositório. Portanto, gerar JSON exportável ou instruir o Antigravity a mesclar o item no arquivo `biblioteca_catalogo.json`.
