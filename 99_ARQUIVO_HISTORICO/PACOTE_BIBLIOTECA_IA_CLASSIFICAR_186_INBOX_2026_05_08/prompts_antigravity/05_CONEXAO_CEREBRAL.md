# 05 — Criar conexão cerebral

Para cada arquivo catalogado:

1. Criar node no grafo.
2. Criar edge para `biblioteca-ia`.
3. Criar edge para tema principal.
4. Criar edges para temas correlatos.
5. Conectar a:
   - Banco TEMI se gerar questão;
   - Card Feed se gerar imagem/card;
   - Calculadoras UTI se tiver dose/cálculo;
   - módulo clínico se complementar tema existente.
