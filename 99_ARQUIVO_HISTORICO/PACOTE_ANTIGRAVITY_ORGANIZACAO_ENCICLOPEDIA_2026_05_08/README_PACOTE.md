## Como usar

1. Descompacte este ZIP.
2. Copie a pasta para dentro do seu repositório local no iCloud.
3. Abra o repositório no Antigravity.
4. Cole o conteúdo de:

```text
prompts_antigravity/00_SUPERCOMANDO_RENDERIZAR_TUDO.md
```

5. Peça ao Antigravity para:
   - integrar os arquivos;
   - renderizar catálogo;
   - validar links;
   - atualizar cards;
   - gerar relatório final.
