# 📌 Resumo por conceitos importantes — Autoanticorpos no LES

## 🧭 Conceito-mãe
Autoanticorpo no LES deve ser interpretado com **clínica + método + tendência + complemento + urina + órgão-alvo**.

## 🧬 Anti-ssDNA
- Baixa especificidade.
- Pouca utilidade para diagnóstico e atividade.
- **Memória:** ssDNA = “sem segurança”.

## 🧪 Anti-dsDNA
- Ajuda no diagnóstico de LES.
- Pode acompanhar atividade em alguns pacientes.
- Associação clássica com nefrite lúpica.
- **Memória:** dupla fita = duplo valor: diagnóstico + rim.

## 🫘 Alerta renal
```text
anti-dsDNA ↑ + C3/C4 ↓ + proteinúria/sedimento ativo = pensar em nefrite lúpica ativa
```

## 🔬 Método importa
- **Crithidia:** mais específico, bom para confirmação.
- **ELISA:** mais quantitativo/sensível, bom para seguimento, mas menos específico.
- **Regra:** Crithidia confirma; ELISA acompanha; tendência decide.

## 🔖 Anti-Sm
- Baixa sensibilidade.
- Alta especificidade.
- Pode persistir na remissão.
- **Memória:** Sm = selo marcado de LES.

## 🧩 Anti-U1 RNP
- Pode ocorrer no LES.
- Alto título sugere DMTC/sobreposição.
- **Memória:** RNP = raciocine na possibilidade de sobreposição.

## 🎯 Pegadinhas
- Anti-Sm negativo não exclui LES.
- Anti-dsDNA negativo não exclui LES nem nefrite.
- Anti-U1 RNP não é específico de LES.
- Anti-ssDNA não deve guiar conduta importante.
- Anti-dsDNA isolado não fecha nefrite: olhe urina, complemento e rim.
