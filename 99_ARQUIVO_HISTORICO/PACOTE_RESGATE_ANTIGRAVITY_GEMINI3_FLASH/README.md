# Pacote de Resgate — Antigravity/Gemini 3 Flash

Este pacote foi gerado para orientar a correção do repositório `aldenirfilho/antigravity-consultas`.

## Como usar

1. Abra o Antigravity no repositório.
2. Cole primeiro `01_SUPERCOMANDO_GEMINI_3_FLASH_RESGATE.md`.
3. Peça ao Gemini para executar em **Planning Mode**.
4. Só depois peça para aplicar por fases:
   - Biblioteca;
   - UpDown;
   - Mapa Vivo;
   - Calculadoras;
   - validação.
5. Use os manifests JSON como base para reconstruir os catálogos.
