Diagnóstico de LES
Critérios EULAR/ACR 2019

Algoritmo passo a passo para classificação do Lúpus Eritematoso Sistêmico — pronto para plantão, UTI e TEMI

// Início
🩺 Suspeita clínica de LES
Mulher jovem (20–40 anos) com ≥2 dos seguintes: artrite simétrica · erupção malar · úlceras orais indolores · serosite · leucopenia · Raynaud · proteinúria inexplicada
Solicitar exame
// Passo 01 · Critério de Entrada
🧪 ANA ≥ 1:80 em HEp-2?
Preferencialmente por imunofluorescência indireta (IFI) em células HEp-2.
Ensaios em fase sólida também são aceitos com desempenho equivalente.

⚠️ Se ANA negativo por fase sólida: repetir por IFI antes de excluir LES.
⚠️ ~5% dos LES são ANA-negativos — especialmente anti-Ro/SSA isolado.
✅ SIM → Continuar
❌ NÃO → Não classificar como LES
// Passo 02 · Workup completo
📋 Solicitar painel completo
Hemograma (leucopenia, trombocitopenia, hemólise) · Creatinina + EAS com sedimento + proteína/creatinina · C3, C4 · Anti-dsDNA · Painel ENA (anti-Sm, anti-RNP, anti-Ro/SSA, anti-La/SSB) · Antifosfolipídios (ACL IgG/M, anti-β2GP1, anticoagulante lúpico) · VHS + PCR · TSH + anti-TPO
Calcular pontuação
// Passo 03 · Pontuação aditiva

≥ 10 pts

Com ANA positivo e ao menos 1 critério clínico. Iniciar tratamento.

🔍

DITC ou outro diagnóstico diferencial. Investigar alternativas.
✅ Se LES confirmado
// Passo 05 · Conduta imediata
💊 Iniciar Hidroxicloroquina AGORA
Todos os pacientes com LES devem receber hidroxicloroquina independente da gravidade.
Dose: 5 mg/kg/dia de peso real (máx. 400 mg/dia) · início do efeito em ~3 meses.
Escalonar terapia conforme gravidade: leve → AINE/prednisona baixa dose | moderada → imunossupressor | grave → pulso + ciclofosfamida/micofenolato.
Monitoramento contínuo
