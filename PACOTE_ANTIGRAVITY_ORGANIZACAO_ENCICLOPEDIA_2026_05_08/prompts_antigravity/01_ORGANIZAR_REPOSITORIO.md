# 01 — Organizar Repositório

Faça auditoria completa do repositório e gere uma árvore de arquivos.

Classifique cada arquivo como:

- portal;
- módulo clínico;
- banco de questões;
- calculadora;
- biblioteca;
- card;
- imagem;
- dashboard;
- dados JSON;
- assets;
- documentação;
- script;
- desconhecido.

Depois atualize:

- `CATALOGO_PROJETO.md`
- `data/catalogo.json`

Não apague nada.
Não mova arquivos ativos sem corrigir links.
