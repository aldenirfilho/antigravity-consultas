# 02 — Classificar Inbox do Card Feed

Analise:

`06_Card_Feed_Medico/assets/cards/inbox/`

e:

`06_Card_Feed_Medico/data/inbox.json`

Para cada imagem:

1. Abrir visualmente.
2. Identificar tema.
3. Criar nome em snake_case sem acentos.
4. Mover para a pasta correta de `themes.json`.
5. Criar entrada em `cards.json`.
6. Remover do `inbox.json` se classificada.
7. Manter como `revisar` se duvidosa.

Temas permitidos:

- inbox
- uti-geral
- temi
- vm-sdra
- sepse-choque
- nefro-aki-trs
- endocrino-diabetes
- neuro-uti
- cardio-hemodinamica
- infectologia

- familia-paciente
- ia-produtividade
