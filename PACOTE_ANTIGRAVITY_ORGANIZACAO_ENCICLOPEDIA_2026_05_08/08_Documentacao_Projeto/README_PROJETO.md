# README do Projeto — Enciclopédia Médica Intensiva & Medicina Interna

## Missão

Transformar conteúdos de UTI, Clínica Médica, TEMI, artigos, documentos, imagens e análises de IA em uma enciclopédia navegável, didática e útil para plantão, estudo e decisão clínica.

## Estrutura editorial

Cada tema deve evoluir para um hub contendo:

- fisiopatologia;
- diagnóstico;
- estratificação de risco;
- conduta;
- UTI;
- prescrição e doses;
- complicações;
- checklist;
- pontos TEMI;
- flashcards;
- imagens/cards;
- referências;
- conexões com temas relacionados.

## Segurança clínica

O conteúdo é educacional e de apoio à decisão.  
Deve preservar julgamento médico, protocolos locais, individualização e dupla checagem em drogas/procedimentos de risco.
