# UPDOWN_001_LES_Manifestacoes_Diagnostico

Pacote pronto para adicionar ao diretório do projeto **Enciclomedia / Antigravity**.

## Conteúdo da pasta

- `index.md` — página inicial curta do módulo
- `updown_001_les_manifestacoes_diagnostico.md` — conteúdo completo em Markdown
- `questoes_temi_r3_les.md` — 10 questões comentadas
- `flashcards_les.md` — memory cards
- `sugestoes_imagens_les.md` — fila de imagens/infográficos sugeridos
- `fluxograma_mermaid_les.md` — fluxograma em Mermaid
- `antigravity_instructions.md` — instruções para indexação/linkagem
- `metadata.json` — metadados para busca e relacionamento automático

## Como usar

1. Descompacte esta pasta.
2. Copie a pasta inteira para o diretório do seu projeto, por exemplo:

```text
/enciclomedia/conteudos/reumatologia/UPDOWN_001_LES_Manifestacoes_Diagnostico/
```

3. Aponte o link da página principal para:

```text
UPDOWN_001_LES_Manifestacoes_Diagnostico/index.md
```

4. Use as tags do `metadata.json` para indexar e relacionar automaticamente com módulos futuros.
