# Instruções para Antigravity / Enciclomedia

## Objetivo
Adicionar o módulo UPDOWN #001 ao site como uma página em modo leitor, sem exibir Markdown bruto.

## Ação principal
Copie a pasta inteira para o diretório desejado do projeto e crie o link da página inicial apontando para:

```text
UPDOWN_001_LES_Manifestacoes_Diagnostico_SITE_READER/index.html
```

## Regra importante
Não linkar diretamente para `.md` se o site não tiver renderizador Markdown configurado.
O arquivo `index.html` já está pronto para exibição pública.

## Sugestão de card para a homepage
Título: UPDOWN #001 — LES em adultos  
Descrição: Manifestações clínicas e diagnóstico do lúpus eritematoso sistêmico em adultos.  
Tags: LES, Reumatologia, Clínica Médica, UTI, TEMI, R3.
