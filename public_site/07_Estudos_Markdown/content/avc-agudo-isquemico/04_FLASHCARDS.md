# 🧠 Flashcards — AVC Agudo

1. **Qual exame inicial exclui hemorragia?** → TC de crânio sem contraste.
2. **Dose da alteplase no AVCi?** → 0,9 mg/kg, máximo 90 mg; 10% bolus e 90% em 60 min.
3. **Dose da tenecteplase no AVCi?** → 0,25 mg/kg EV bolus, máximo 25 mg.
4. **PA antes da trombólise?** → <185/110 mmHg.
5. **PA após trombólise?** → <180/105 mmHg por 24h.
6. **Pode dar AAS logo após alteplase?** → Não. Aguardar 24h e imagem.
7. **Hipoglicemia pode simular AVC?** → Sim.
8. **O que é penumbra?** → Tecido hipoperfundido potencialmente recuperável.
9. **O que é LVO?** → Oclusão de grande vaso.
10. **Exame para detectar LVO?** → Angio-TC ou angio-RM.
11. **Basilar pode ter clínica flutuante?** → Sim, e pode ser devastador.
12. **Sinal de AVC cerebelar grave?** → Rebaixamento, vômitos, hidrocefalia, compressão de tronco.
13. **Principal risco pós-trombólise?** → Hemorragia sintomática.
14. **Conduta se piora pós-trombólise?** → TC urgente e manejo de sangramento.
15. **INR limite clássico para alteplase?** → Evitar se >1,7.
16. **Plaquetas mínimas para trombólise?** → ≥100.000/mm³.
17. **Dissecção de aorta suspeita?** → Não trombolisar.
18. **HSA suspeita?** → Não trombolisar.
19. **Wake-up stroke pode tratar?** → Sim, se seleção por imagem e protocolo.
20. **HIC em varfarina?** → PCC + vitamina K EV.
21. **Dabigatrana com HIC?** → Idarucizumabe se disponível.
22. **Heparina com sangramento grave?** → Protamina.
23. **ICH Score serve para quê?** → Prognóstico/comunicação de gravidade.
24. **ABCD2 serve para quê?** → Risco após AIT.
25. **TOAST classifica quê?** → Etiologia do AVCi.
26. **ESUS significa anticoagular automaticamente?** → Não.
27. **Dieta pós-AVC?** → Só após avaliação de deglutição.
28. **Febre no AVC?** → Tratar ativamente.
29. **Hipotensão no AVCi?** → Evitar; reduz perfusão da penumbra.
30. **Frase-chave?** → Tempo é cérebro, fisiologia é destino.
