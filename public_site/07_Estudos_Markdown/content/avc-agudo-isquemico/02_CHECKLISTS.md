# 📋 Checklists de Segurança no AVC Agudo

## Código AVC em 10 minutos

- [ ] Acionar equipe/protocolo AVC.
- [ ] ABCDE.
- [ ] Glicemia capilar.
- [ ] Último visto bem.
- [ ] NIHSS inicial.
- [ ] PA, FC, SpO₂, temperatura.
- [ ] Anticoagulantes? Qual? Última dose?
- [ ] TC sem contraste.
- [ ] Angio-TC se suspeita de LVO.
- [ ] Decidir trombólise/trombectomia/transferência.

## Checklist pós-trombólise

- [ ] PA <180/105.
- [ ] Monitorização neurológica seriada.
- [ ] Sem AAS/anticoagulante por 24h.
- [ ] Evitar punções invasivas desnecessárias.
- [ ] Vigiar angioedema, sangramento, cefaleia, vômitos, piora neurológica.
- [ ] TC em 24h ou antes se piora.
- [ ] Deglutição antes de dieta.

## Checklist UTI

- [ ] Via aérea segura?
- [ ] SpO₂ adequada?
- [ ] PaCO₂ aceitável se ventilado?
- [ ] PA dentro da meta?
- [ ] Glicemia controlada?
- [ ] Febre tratada?
- [ ] Pupilas/Glasgow/NIHSS seriados?
- [ ] Sinais de edema/herniação?
- [ ] Profilaxia TVP adequada?
- [ ] Nutrição segura?
- [ ] Investigação etiológica iniciada?
- [ ] Família atualizada?

## Checklist HIC

- [ ] Localização e volume.
- [ ] Extensão intraventricular.
- [ ] Hidrocefalia.
- [ ] Anticoagulante/antiagregante.
- [ ] Reversão indicada e feita?
- [ ] PA controlada sem queda brusca?
- [ ] Neurocirurgia acionada se cerebelar/IVH/hidrocefalia/deterioração?
- [ ] UTI indicada?
