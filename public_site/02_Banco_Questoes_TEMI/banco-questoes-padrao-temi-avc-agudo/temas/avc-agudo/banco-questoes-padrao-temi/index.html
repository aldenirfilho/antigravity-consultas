🕸️ Como funciona a linkagem automática

Cada questão possui um campo related com slugs de temas correlacionados. O JavaScript cruza esses slugs com o topicRegistry. Quando o módulo existir no hub, mude o status para ativo e ajuste o path. O chip passa a funcionar como link real.

  • source: questão TEMI.
  • target: tema correlacionado.
  • relation: aborda, depende-de, reforça ou revisa.
  • autoLink: permite que o grafo do hub leia o relacionamento.
