# 📥 Inbox — Depósito Geral da Biblioteca IA

Jogue aqui **TODOS** os seus PDFs, Word (.docx) e HTMLs de uma vez.

## Formatos aceitos
- `.pdf` — Artigos, guidelines, resumos
- `.docx` / `.doc` — Documentos Word
- `.html` — Páginas interativas

## Fluxo
1. Cole todos os arquivos nesta pasta
2. Rode: `cd 05_Biblioteca_IA && bash scan_inbox.sh`
3. Faça commit + push
4. Abra a Biblioteca no navegador
5. Clique em **📂 Classificar** em cada arquivo
6. Escolha o tema → o arquivo é categorizado

## ⚠️ Segurança
- NÃO suba documentos com dados de pacientes identificáveis
- NÃO suba prontuários, prescrições reais ou exames com nome
