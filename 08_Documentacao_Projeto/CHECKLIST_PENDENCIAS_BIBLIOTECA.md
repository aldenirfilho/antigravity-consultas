# Checklist — Pendências Biblioteca IA

## Duplicados

- [ ] Criar `biblioteca_duplicados.json`.
- [ ] Marcar confiança alta/média/baixa.
- [ ] Não apagar automaticamente.
- [ ] Gerar relatório.

## Cards

- [ ] Selecionar alto rendimento.
- [ ] Criar candidatos.
- [ ] Conectar ao arquivo origem.
- [ ] Não gerar imagem sem revisão.
- [ ] Criar patch para Card Feed.

## Questões TEMI

- [ ] Selecionar materiais TEMI.
- [ ] Criar candidatos.
- [ ] Definir dificuldade.
- [ ] Criar comentário.
- [ ] Criar pérola TEMI.
- [ ] Gerar patch separado.

## Conexões cerebrais

- [ ] Criar nodes.
- [ ] Criar edges.
- [ ] Conectar módulos existentes.
- [ ] Criar patches para módulos planejados.

## Validação

- [ ] JSON válido.
- [ ] Sem IDs duplicados.
- [ ] Links funcionando.
- [ ] Caminhos existem.
- [ ] Interface carrega.
- [ ] Relatório final preenchido.
