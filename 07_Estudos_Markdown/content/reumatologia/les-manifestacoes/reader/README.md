# UPDOWN #001 — LES em adultos: manifestações clínicas e diagnóstico

Esta pasta contém a versão **pública em modo leitor** do módulo.

## Arquivo correto para linkar no site

Use sempre:

```text
index.html
```

Não use `public_content.md` como página pública, porque muitos sites exibem Markdown como texto bruto quando não existe pipeline de renderização.

## Recursos do modo leitor

- HTML já renderizado
- Sumário lateral
- Modo claro/escuro
- Controle de tamanho da fonte
- Botão de imprimir/salvar em PDF
- Layout responsivo para celular
- Suporte a Mermaid para fluxogramas

## Conteúdo público

O documento público contém apenas o conteúdo educacional do UPDOWN.
Não contém sugestões internas de imagens, prompts de produção ou bastidores do projeto.
