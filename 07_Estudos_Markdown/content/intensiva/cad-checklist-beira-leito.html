⚡ REGRA DE OURO: Nunca inicie insulina sem checar K⁺. Se K⁺ < 3,3 → repor KCl primeiro. A acidose resolve mais devagar que a hiperglicemia — não suspenda insulina EV só porque a glicemia normalizou.
⚡ 0–15 min · Admissão
✓
Acesso venoso calibroso (16G+) 2 acessos se grave
✓
Gasometria arterial urgente pH, HCO₃⁻, PCO₂, lactato
✓
Coleta de K⁺, Na⁺, Cl⁻, glicemia Antes de qualquer insulina
✓
Cetonúria (fita) ou beta-OH-butirato
✓
ECG (12 derivações)
✓
Monitor cardíaco contínuo
✓
Oximetria de pulso
✓
Sonda vesical (se grave/comatoso)
💧 Hidratação EV
✓
SF 0,9% — 1 L na 1ª hora 15–20 mL/kg/h (hipovolemia)
✓
2ª–4ª hora: 500 mL/h SF 0,9% ou 0,45%
✓
Se Na⁺ corrigido alto → SF 0,45%
✓
Glicemia < 250: adicionar SG 5% Manter glicemia 150–200 mg/dL
✓
Balanço hídrico horário
✓
Diurese alvo ≥ 0,5 mL/kg/h
⚠️ Não reidratar rápido demais → risco de edema cerebral (especialmente pediátrico)
⚡ Protocolo de Potássio — CRÍTICO
K⁺ sérico Insulina EV Reposição KCl Monitoração
< 3,3 mEq/L ⛔ NÃO iniciar KCl 20–40 mEq/h EV K⁺ a cada 1h
3,3–5,0 mEq/L ✅ Iniciar KCl 20–30 mEq/L hidratação K⁺ a cada 2h
> 5,0 mEq/L ✅ Iniciar Sem KCl ECG contínuo · K⁺ 2/2h
💉 Insulina EV
✓
K⁺ ≥ 3,5 confirmado antes de iniciar
✓
Preparar: 100 U insulina regular em 100 mL SF 0,9% (1 U/mL)
✓
Infusão: 0,1 U/kg/h Em bomba de seringa
✓
Glicemia capilar horária Meta: queda 50–75 mg/dL/h
✓
Queda < 50 mg/dL/h → dobrar taxa
✓
Glicemia < 250: reduzir para 0,02–0,05 U/kg/h + SG5%
✓
Não suspender EV até pH > 7,30 + HCO₃⁻ > 18 + AG < 12
