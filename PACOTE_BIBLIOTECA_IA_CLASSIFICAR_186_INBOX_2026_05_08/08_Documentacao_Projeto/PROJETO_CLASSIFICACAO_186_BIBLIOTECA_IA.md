# Projeto — Classificação de 186 arquivos do Inbox da Biblioteca IA

## Objetivo

Criar uma esteira segura para classificar manualmente 186 arquivos do inbox da Biblioteca IA usando a interface/edição assistida do Antigravity.

## Entregas esperadas

1. Escanear `05_Biblioteca_IA/inbox/`.
2. Preencher `biblioteca_inbox_manifest_186.json` com nomes reais.
3. Classificar cada arquivo por tema.
4. Mover para pasta final.
5. Renomear com padrão seguro.
6. Atualizar `biblioteca_catalogo.json`.
7. Atualizar `biblioteca_brain_connections.json`.
8. Atualizar a página `05_Biblioteca_IA/index.html` para consumir os dados.
9. Gerar relatório final.

## Conceito de conexão cerebral

Cada arquivo não deve ser só guardado em uma pasta. Ele deve virar um nó no grafo de conhecimento, conectado a:

- tema principal;
- temas correlatos;
- módulos clínicos;
- Banco TEMI;
- Card Feed;
- Calculadoras UTI;
- possíveis futuros protocolos.

## Fluxo manual ideal

Para cada arquivo:

1. Abrir preview.
2. Identificar assunto.
3. Selecionar tema.
4. Definir título didático.
5. Gerar tags.
6. Escrever resumo curto.
7. Marcar relevância TEMI.
8. Definir uso prático.
9. Salvar no catálogo.
10. Mover/renomear arquivo.
