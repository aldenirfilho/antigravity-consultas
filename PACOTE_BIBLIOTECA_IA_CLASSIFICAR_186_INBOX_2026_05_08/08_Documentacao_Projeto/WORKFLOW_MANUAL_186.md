# Workflow Manual — 186 arquivos Inbox Biblioteca IA

## Lotes recomendados

Classificar em lotes de 20 arquivos:

- Lote 01: arquivos 001–020
- Lote 02: arquivos 021–040
- Lote 03: arquivos 041–060
- Lote 04: arquivos 061–080
- Lote 05: arquivos 081–100
- Lote 06: arquivos 101–120
- Lote 07: arquivos 121–140
- Lote 08: arquivos 141–160
- Lote 09: arquivos 161–180
- Lote 10: arquivos 181–186

## Status por arquivo

- `inbox`: ainda não aberto.
- `em-triagem`: aberto, mas não finalizado.
- `catalogado`: classificado, renomeado e movido.
- `revisar`: assunto duvidoso.
- `duplicado`: duplicado provável.
- `arquivado`: não publicar.
- `publicado`: já visível na Biblioteca.

## Regra TDAH-friendly

Não tentar fazer 186 de uma vez.  
Usar lotes curtos, salvar sempre e validar depois de cada lote.
