# 04 — Classificar lote de 20 arquivos

Escolha o próximo lote de 20 arquivos com `status: "inbox"`.

Para cada arquivo:

1. Abrir preview.
2. Identificar tema.
3. Preencher metadados.
4. Renomear.
5. Mover para pasta final.
6. Atualizar `biblioteca_catalogo.json`.
7. Atualizar `biblioteca_brain_connections.json`.
8. Atualizar status no manifest.

Se dúvida: status `revisar` e manter no inbox.
