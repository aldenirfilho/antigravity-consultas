# Convenção de nomes — Biblioteca IA

## Padrão

```text
tema_subtema_descricao_curta_ano.ext
```

## Exemplos

```text
vm_sdra_ventilacao_protetora_2026.pdf
nefro_aki_teste_estresse_furosemida_uti_2026.docx
infectologia_cvc_staphylococcus_aureus_conduta_2026.pdf
endocrino_cetoacidose_diabetica_uti_2026.html
neuro_uti_avc_agudo_codigo_10_minutos_2026.pdf
```

## Regras

- usar minúsculas;
- sem acentos;
- sem espaços;
- usar `_`;
- manter extensão original;
- se for versão revisada, adicionar `_v2`;
- se for material bruto, adicionar `_bruto`;
- se precisar revisão, adicionar `_revisar`.
