📌 Como adicionar seus arquivos PDF/Word à Biblioteca

  1. GitHub (recomendado): Suba os arquivos na pasta biblioteca_ia/arquivos/ do seu repositório GitHub (aldenirfilho/antigravity-consultas)
  2. Após upload, copie o link direto: https://aldenirfilho.github.io/antigravity-consultas/biblioteca_ia/arquivos/NOME.pdf
  3. Adicione o card correspondente no array ARQUIVOS abaixo no JavaScript desta página
  4. Google Drive (alternativa): Compartilhe o arquivo como "qualquer pessoa com o link pode visualizar" e use o ID do Drive no link de embed
  5. Para PDFs do Drive: https://drive.google.com/file/d/ID_DO_ARQUIVO/preview

💡 Dica TDAH: Use uma planilha simples no Google Sheets para registrar título, link, tags e data de cada arquivo antes de adicionar aqui. Facilita a manutenção!
