# Card Feed Médico — Pack de Integração Antigravity

Este pacote contém o módulo pronto para ser copiado para o repositório:

`aldenirfilho/antigravity-consultas`

Arquivos principais:

- `06_Card_Feed_Medico/index.html`: app/feed integrado.
- `06_Card_Feed_Medico/data/cards.json`: banco inicial de cards.
- `06_Card_Feed_Medico/data/themes.json`: temas iniciais.
- `PATCH_INDEX_HOME.html`: trechos para inserir na home.
- `ANTIGRAVITY_INTEGRACAO_CARD_FEED.md`: instruções completas.
- `PROMPT_ANTIGRAVITY_ORGANIZAR_CARD_FEED.txt`: prompt pronto para colar no Antigravity.

Fluxo:
1. Copie a pasta `06_Card_Feed_Medico` para a raiz do repo.
2. Aplique os patches em `index.html`.
3. Suba imagens para `assets/cards/<tema>/`.
4. Atualize `data/cards.json`.
5. Publique no GitHub Pages.
