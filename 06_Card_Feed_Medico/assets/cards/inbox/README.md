# 📥 Inbox — Depósito geral de cards

Jogue aqui TODAS as imagens/cards de uma vez, sem se preocupar com classificação.

## Como funciona

1. Cole todas as imagens PNG/JPG nesta pasta
2. Rode o script `scan_inbox.sh` para gerar o manifesto
3. Abra o Card Feed no navegador
4. Clique no botão **📂 Classificar** em cada card
5. Escolha o tema correto → o card é reclassificado automaticamente

## Padrão de nomes (recomendado, mas não obrigatório)

```
assunto_detalhe_1080x1920.png
```

Pode jogar com qualquer nome — o app sugere tema automaticamente pelo nome do arquivo.

## Segurança

- ⚠️ NÃO suba imagens com dados identificáveis de pacientes
- ⚠️ NÃO suba fotos de prontuário, CPF, RG, exames com nome
