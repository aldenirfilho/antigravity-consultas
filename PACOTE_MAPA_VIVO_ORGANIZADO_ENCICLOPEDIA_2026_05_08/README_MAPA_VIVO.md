# Pacote — Mapa Vivo Organizado da Enciclopédia

**Data:** 2026-05-08  
**Projeto:** Enciclopédia Médica Intensiva & Medicina Interna  
**Objetivo:** transformar o Mapa Vivo em um cérebro visual com nós, temas, módulos e conexões clínicas.

## Arquivos prontos para aplicar

```text
data/topics.json
data/connections.json
assets/js/graph.js
assets/css/graph-vivo.css
```

## Como usar no Antigravity

1. Copie este pacote para o repositório.
2. Cole o prompt:
   `prompts_antigravity/00_SUPERCOMANDO_ORGANIZAR_MAPA_VIVO.md`
3. Peça para aplicar os arquivos, validar a home e testar o grafo.

## O que muda

- Expande `topics.json`.
- Expande `connections.json`.
- Adiciona nós para Biblioteca IA, Card Feed, Banco TEMI, Calculadoras, AVC e macrotemas.
- Atualiza `graph.js` para desenhar arestas/linhas.
- Cria CSS específico para o mapa.
- Mantém links relativos.
