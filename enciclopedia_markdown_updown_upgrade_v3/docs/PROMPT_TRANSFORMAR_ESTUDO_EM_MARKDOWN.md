# 🧠 Prompt para transformar estudo em Markdown seguro

Use este prompt no Gemini Flash, GPT, Claude ou outra IA.

```text
Transforme minhas notas de estudo em um documento Markdown para a Enciclopédia Médica.

Regras obrigatórias:
1. Não copie texto literal extenso de fonte protegida.
2. Reescreva com linguagem própria, didática e clínica.
3. Gere material útil para UTI, Clínica Médica e prova TEMI.
4. Organize em Markdown com frontmatter.
5. Inclua versão, status, tema, tags e conexões.
6. Separe: definição, fisiopatologia, quadro clínico, diagnóstico, conduta, UTI, beira-leito, pegadinhas TEMI, checklist e produtos derivados.
7. Quando houver informação extra, marque como 🔎 Informação extra.
8. Gere também sugestões de flashcards, questões e cards visuais.
9. Não mencione fontes proprietárias nominalmente no título público.
10. Finalize com pendências de revisão e assuntos correlatos.

Minhas notas:
[COLE AQUI SUAS NOTAS, SEM COLAR CAPÍTULO INTEIRO]
```
